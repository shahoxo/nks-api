--- layout: default title: NKS API description: API documentation for nks-api.shahoxo.com theme: jekyll-theme-hacker --- # NKS API Welcome to the NKS API documentation. This site provides information about the API endpoints and how to use them. ## Overview The NKS API provides race prediction data for horse racing events. You can access race data by specifying a race ID. ## Getting Started To use the API, simply make a GET request to the endpoint with the race ID. ## API Endpoints ### Race Data Endpoint: `/api/v1/{race_id}.json` Example: `/api/v1/202508020503.json` ## Try It Out
